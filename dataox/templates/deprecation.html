Deprecation of data.ox.ac.uk

Data.ox.ac.uk has been deprecated. This page offers information and advice of how you can replace it.

Oxpoints

Location of data for University buildings can now be found at www.maps.ox.ac.uk

Research Equipment

The new Research Equipment site is https://researchsupport.web.ox.ac.uk/facilities/

Vacancies

Instead of using data.ox.ac.uk we recommend getting XML directly from CoreHR. Vacancies can be found by using the appropriate PRAS Unit or Department Code.

Unit codes are at the following location of the XML heirarchy: currentVacancies > vacancy > managementUnit > code = "YOUR UNIT CODE"

Departemnt codes are at the following location of the XML heirarchy: currentVacancies > vacancy > department > code = "YOUR DEPT CODE"

We recommend that you cache the data as CoreHR's XML is large and therefore can take a while to load.
